# Crime Prediction Using Machine Learning

## Overview
This project predicts crime occurrence using demographic factors such as population, unemployment rate, and poverty rate.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-Learn
- Random Forest Classifier

## Features
- Data Preprocessing
- Model Training
- Accuracy Evaluation
- Crime Prediction

## Run Project

pip install -r requirements.txt

python train_model.py

python predict.py
